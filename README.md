# Dombyra.kz — Мастерская домбры ручной работы

Премиальный веб-сайт мастерской по изготовлению казахских национальных инструментов. Проект сочетает в себе традиционную эстетику и современные технологии веб-разработки.

## 🚀 Технологии

*   **Frontend**: Tailwind CSS (CDN), Vanilla JavaScript.
*   **Backend**: PHP (Modular architecture).
*   **Design**: Респонсивный дизайн, анимации на SVG, интерактивная карта Казахстана.

## 📂 Структура проекта

*   `index.php` — Главная страница с конструктором и героями.
*   `catalog.php` — Каталог инструментов.
*   `articles.php` — База знаний и статьи.
*   `contacts.php` — Контакты и интерактивная карта.
*   `includes/` — Переиспользуемые PHP-компоненты (шапка, подвал, модули).
*   `assets/` & `img/` — Медиа-ресурсы и стили.

## 🛠 Локальная установка

Для запуска проекта на локальном компьютере необходим PHP сервер (PHP 7.4+).

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/YOUR_USERNAME/dombyra.git
    ```
2.  Запустите локальный сервер. Если у вас установлен PHP, выполните команду в корневой папке:
    ```bash
    php -S localhost:8000
    ```
3.  Откройте `http://localhost:8000` в браузере.
